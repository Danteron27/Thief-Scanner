# Instructivo de Uso para la Aplicación de Captura de Tráfico

## Introducción
Este documento proporciona las instrucciones necesarias para utilizar la aplicación de captura de tráfico. Es fundamental seguir cada uno de los pasos a continuación para asegurar un funcionamiento óptimo del programa.

## Requisitos Previos
Antes de ejecutar la aplicación, asegúrate de tener instalado **Npcap**. Esta herramienta es esencial para que la aplicación funcione correctamente, ya que permite la captura de paquetes en la red. Puedes descargar Npcap desde su página oficial:

## Pasos para Ejecutar la Aplicación

1. **Instalación de Npcap**
   - Visita la página de descarga de Npcap y descarga el instalador.
   - Ejecuta el instalador y sigue las instrucciones en pantalla para completar la instalación.
2. **Ejecución del Programa**
   - Abre la carpeta dist
   - Ejecuta la aplicación con nombre captura_trafico.exe
3. **Uso de la Interfaz Gráfica**
   - Una vez abierta la aplicación, selecciona la interfaz de red desde la lista desplegable.
   - Configura el tiempo de escaneo según tus preferencias (1, 3, 5, 10 minutos o indefinido).
   - Haz clic en "Iniciar Captura" para comenzar a monitorear el tráfico de red.

4. **Detención de la Captura**
   - Para detener la captura, haz clic en el botón "Detener Captura".
   - Se mostrará un resumen con el total de paquetes capturados, IP más común, alertas generadas, duración del escaneo y estado general del tráfico.


¡Gracias por utilizar nuestra aplicación de captura de tráfico!
